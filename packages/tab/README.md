
# Future-tab

> A mobile touch-swappable component

 一个用于移动端的可触摸滑动的Vue组件

## Installation
```shell
npm i future-tab -S

```
or
```shell
yarn add future-tab

```

## Usage

```javascript
import Vue from 'vue'
import FutureTab from 'future-tab'

Vue.use(FutureTab)
// 之后便可在全局使用了
```

## Build Setup

```bash
# install dependencies
npm install
# or
yarn

# serve with hot reload at localhost:8080
npm run dev
# or
yarn dev

```
